# ladychrome

ASCII art for DJ Lady Chrome. Three self-contained pages, no build step.

- `/` — chrome figure (the brand image converted to glowing ASCII)
- `/disco-ball` — iridescent rhinestone disco ball
- `/record-player` — ASCII turntable

Each page measures the device's monospace glyph ratio at runtime to size and
proportion the art on any screen.

Live: https://ladychrome.vercel.app
